# Prueba_Tuya
pruebeingeniero de datos

Ejercicio 1 - Dataset Teléfonos:

Se diseña un pipeline en Microsoft Fabric usando Data Factory para ingesta desde fuentes de datos como un CRM a una capa Bronze, posterior a esto con Power Query se realiza el proceso de ETL a una capa Silver, y despliega el aprovechamiento de los metadatos a una capa de producción Gold).Adicional se Implementa CI/CD via con noteboocks de pyspark con pruebas de calidad como nulos o duplicados 0% y luego un despliegue automático al lakehouse.


Ejercicio 2 - KPI's Calidad
Se puede usar Microsoft Purview para scaneos automáticos de calidad.
Creación de Dashboards en Power BI con DAX para KPIs.
Churn para obtener un porcentaje datos inválidos o también para medir la perdida de clientes 
Trazabilidad y disponibilidad de los datos.
Aprovechamiento de la información para reportes, Dashboards Power bi e IA.
